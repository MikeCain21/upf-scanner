UPF Scanner - Test Harness

Version: 0.1.0 | Phase: 1 - Extension Setup

📋 Testing Instructions

  1. Open Chrome and navigate to chrome://extensions/
  2. Enable Developer mode (toggle in top-right corner)
  3. Click "Load unpacked" and select the upf-scanner folder
  4. Verify the extension appears in the list with no errors
  5. Open this page (test-runner.html) in a new tab
  6. Open Developer Tools (F12 or right-click → Inspect)
  7. Go to the Console tab
  8. Check for extension logs (see Expected Results below)
⚠️ Important: This test harness simulates a basic page structure. The extension content script is configured to run only on tesco.com domains in the manifest. For Phase 1 testing, you may need to temporarily modify the manifest to test on this local file, OR wait until Phase 2 when we have actual Tesco HTML pages.
